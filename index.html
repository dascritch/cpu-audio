Why ?

Global page controller

↓ Controller only on the playing / last played <audio>

Keyboard controls

  • Space : play/pause audio
  • Enter : play/pause audio (only on play/pause button)
  • ← : -5 seconds
  • → : +5 seconds
  • ↖ : back to start
  • End : to the end, finish playing
  • Escape : back to start, stop playing

Error test-cases

↓ No media tag. The component must not start
↓ Media tag without controls. The component must not start
↓ Bad media call. Alas, Chrome won't crash it until any interaction, Firefox never display any error.
